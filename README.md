
# FocusFinder  ‚ú®


FocusFinder is a web application that helps users find a suitable camera based on their preferences, such as skill level, intended use, camera type, and budget.
The application combines a JavaScript front-end with a Node.js + Express backend and a MongoDB database.

Website URL: https://ehb-mct.github.io/web2-course-project-front-end-SafouanKerkich-1/
